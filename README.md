# 🎓 Agentro - AI College Admission Assistant

Agentro is an AI-powered College Admission Assistant that helps students discover colleges, understand eligibility criteria, compare courses, and receive personalized admission guidance through a modern conversational interface.

> Making college admissions faster, simpler, and more accessible.

---

## ✨ Features

- 🤖 AI-powered admission guidance
- 🏫 College & course recommendations
- 📋 Eligibility checking
- 💬 Natural language chatbot
- 📚 Admission process guidance
- 📄 Required documents information
- 💰 Fee and scholarship information
- 🔍 College search & filtering
- 📱 Responsive UI
- ⚡ Fast and intuitive user experience

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/Agentro.git

cd Agentro
```

---

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

## Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

DATABASE_URL=YOUR_DATABASE_URL

GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## Run Locally

### Backend

```bash
cd backend
npm run dev
```

### Frontend

```bash
cd frontend
npm run dev
```
